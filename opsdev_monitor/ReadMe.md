## 思路
> 五条线

* 每分钟产生一个Ticket --> 加入任务队列 --> 结束
* 死循环 --> 获取Ticket --> 产生采集任务 --> 采集 --> 分析 --> 对比阈值 --> 产生性能数据 --> 产生缓存告警 --> 结束
* 每小时检查一次缓存告警 --> 产生报表数据 --> 发送通知（邮件，短信）--> 记录通知信息 --> 结束
* 每天检查一次 --> 清理日志文件 --> 结束
* 每周检查一次 --> 清理过期数据 --> 结束
